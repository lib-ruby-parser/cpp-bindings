◆ scope

Node* lib_ruby_parser::Casgn::scope

Scope where the constant is defined:

  1. Some(Const("A")) for A::B = 1
  2. None if it's defined in the current scope (i.e. A = 1)
  3. Some(Cbase) if it's defined in the global scope (i.e. ::A = 1)
